Nombre Descripción Tipo Valor por defecto Posibles valores (constants)
showNotifications Indica si debe mostrar el numero de notificaciones boolean true
numberNotifications Numero de notificaciones a mostrar number 0
limitNotifications Limite del numero de notificaciones a mostrar, si lo supera muestra el caracter + y el limite number 9
elementToInsert Especifica a que elemento debe agregar la notificiacion, util para elementos internos o con posiciones fixed string undefined
positionTop Permite personalizar la posicion top de la notificación string undefined
positionRight Permite personalizar la posicion right de la notificación string undefined
