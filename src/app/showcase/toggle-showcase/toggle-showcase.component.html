Componente para mostrar un toggle, ideal para valores booleanos.

Ejemplo 1: estandar

{{'Valor :' + value1}}

Ejemplo 2: label

{{'Valor :' + value2}}

Ejemplo 3: Label inline

{{'Valor :' + value3}}

Ejemplo 4: Small

{{'Valor :' + value4}}

Nombre Descripción Tipo Valor por defecto Posibles valores (constants)
label Label del toggle string undefined
Inline Coloca el label y el toggle en linea boolean false
required Indica si el input es obligatorio boolean false
tooltipText Texto del tooltip a mostrar boolean false
small Hace el componente mas pequeño boolean false
Nombre Descripción Valor devuelto
toggled Evento al clickar el toggle boolean
