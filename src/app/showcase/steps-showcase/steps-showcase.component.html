ddr-step

Nombre Descripción Tipo Valor por defecto Posibles valores (constants)
titleStep Titulo del paso string undefined
canGoNext Condición para habilitar el boton de siguiente. any true

ddr-steps

Nombre Descripción Tipo Valor por defecto Posibles valores (constants)
vertical Indica si se muestran los pasos de forma vertical boolean false
canJumpStep Indica si se pueden saltar de un paso a otro boolean false
openAll Indica si todos los pasos estan abiertos, no permite cambiar de pasos. Solo para el modo vertical boolean false
labelNext Label del boton siguiente string undefined
labelPrevious Label del boton previo string undefined
Nombre Descripción Valor devuelto
changeStep Evento al cambiar de paso number
lastStep Evento al cambiar al ultimo paso boolean
Label Descripción
next Label boton siguiente
previous Label boton previo
