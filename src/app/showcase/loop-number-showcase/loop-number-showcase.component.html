Pipe para recorrer con un ngFor de 0 a n sin necesidad de arrays u objetos.

Ejemplo 1: Estandar

@for (i of 10 | loopNumber; track i) {
{{i}}
}

Ejemplo 2: negativos

@for (i of 5 | loopNumber: {start: -5} ; track i) {
{{i}}
}

Ejemplo 3: anidado

@for (i of 10 | loopNumber; track i) { @for (j of 10 | loopNumber; track j) { } }
{{i}} {{j}}

Ejemplo 4: reverse

@for (i of 10 | loopNumber:{reverse: true} ; track i) {
{{i}}
}

Ejemplo 5: step

@for (i of 10 | loopNumber:{step: 2} ; track i) {
{{i}}
}

Ejemplo 6: includeLast

@for (i of 10 | loopNumber:{includeLast: true} ; track i) {
{{i}}
}
Nombre Descripción Tipo Valor por defecto Posibles valores (constants)
number Numero limite, si es negativo se setea a 0 number undefined
-- DdrParamsLoop
Nombre Descripción Tipo Valor por defecto Posibles valores (constants)
start Numero donde inicia. number 0
includeLast Indica si incluye o no el ultimo elemento. number 0
step Paso en el que avanza. number 1
reverse Indica si queremos mostrarlo al reves. boolean false
