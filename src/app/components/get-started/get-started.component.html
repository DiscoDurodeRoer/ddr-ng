¿Que es la biblioteca de componentes de ddr?
Es un conjunto de componentes/servicios reutilizables para nuestros proyectos.
Como instalar la biblioteca ddr
Para instalar la biblioteca ddr, realizar el siguiente comando: 
npm i @ddr/ddr-frontend-library
¿Que debemos incluir en el fichero angular.json?
Añadiremos lo siguiente en styles: 
"node_modules/bootstrap/dist/css/bootstrap.min.css",
"node_modules/@ddr/ddr-frontend-library/resources/themes/ddr-blue.scss",
"node_modules/bootstrap/dist/css/bootstrap.min.css",
"node_modules/bootstrap-icons/font/bootstrap-icons.css",
"src/styles.scss"
