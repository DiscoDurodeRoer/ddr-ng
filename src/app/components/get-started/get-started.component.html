多Que es la biblioteca de componentes de ddr?
Es un conjunto de componentes/servicios reutilizables para nuestros proyectos.
Como instalar la biblioteca ddr
Para instalar la biblioteca ddr, realizar el siguiente comando: 
npm i @ddr/ddr-frontend-library
